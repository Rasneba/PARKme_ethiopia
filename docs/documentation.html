1. Executive Summary

Parkme Ethiopia is a web-based parking-as-a-service (PaaS) platform designed for Addis Ababa, Ethiopia. It enables drivers to find, reserve, and pay for parking spaces in real time, while allowing hosts (parking space owners) to list, manage, and monetize their parking infrastructure.

🇪🇹 Ethiopian Focus

Local addresses, Amharic street names, ETB currency, TeleBirr + CBE payments, and coverage across 14+ neighborhoods in Addis Ababa.

🌐 Interactive Map

MapLibre GL with GraphHopper routing, clustered markers, real-time directions, turn-by-turn navigation, and satellite view toggle.

💰 Digital Wallet

Built-in wallet with instant top-up via TeleBirr or Commercial Bank of Ethiopia. Cashless parking payments with real-time balance tracking.

🏢 Host Dashboard

Earnings analytics, pricing control, availability management, occupancy metrics, and weekly revenue charts for parking space owners.

📱 PWA Ready

Progressive Web App with offline support, installability, and home screen prompt. Works on iOS, Android, and desktop.

🔒 Secure Auth

Cookie-based sessions, bcrypt password hashing, role-based access (driver/host), and 7-day persistent login.

3.2 Data Flow

  1. User opens app — Landing page renders; if session cookie exists, redirects to /app
  2. Authentication — Modal-based login/signup with role selection (driver or host)
  3. Browse spots — GET /api/spots returns 26+ seeded parking spaces sorted by price
  4. Map rendering — MapLibre loads GraphHopper tiles, overlays OSM roads + buildings, clusters markers
  5. Select spot — Bottom sheet appears with spot details, directions, and reserve button
  6. Get directions — GET /api/directions proxies to GraphHopper; route drawn as green line on map with turn-by-turn popup
  7. Book parking — Duration slider, payment method (wallet/TeleBirr), optional coupon; POST /api/bookings creates reservation
  8. Wallet payment — Balance deducted, transaction recorded; gate code generated
  9. Check-in — POST /api/bookings/[id]/check-in activates parking session
  10. Host dashboard — Earnings analytics, pricing control, availability blocks

4. Driver Guide

4.1 Getting Started

As a driver, Parkme Ethiopia helps you find available parking spots near your destination, reserve them in advance, and navigate directly using built-in directions.

Creating an Account

  1. Visit https://prakme-ethiopia.vercel.app
  2. On the landing page, scroll to the registration form or click "Open App"
  3. Select "I need parking" (Driver role)
  4. Enter your full name, email (e.g., yourname@gmail.com), phone number (+251 prefix), and password (min 6 characters)
  5. Click "Create my account"
  6. You will be redirected to the app with a 500 ETB welcome bonus in your wallet
Demo account: Email miki@parkme.et / Password demo1234 (pre-loaded with 500 ETB and an active booking)

4.2 Finding Parking

Search Panel

  • Use the search bar at the top to search by name, address, or neighborhood
  • Results update in real-time as you type (300ms debounce)
  • Results show: parking name, rating (stars), address, distance from you, available spots, and hourly price in ETB

Filters

Tap the filter icon to access:

  • Available only — Hide fully booked spots
  • Covered — Show only covered/garage parking
  • Under 50 ETB/hr — Budget-friendly options

Map View

  • Clustered markers — Zoom in to expand clusters into individual pins
  • Pin colors: Green = available, Red = selected
  • Near Me button — Flies to your current GPS location
  • Satellite button — Toggle between street map and satellite imagery
  • Zoom controls — Native MapLibre +/- buttons at top-right

Getting Directions

  1. Tap a parking spot (on map pin or search result)
  2. Bottom sheet shows spot details
  3. Tap "Directions"
  4. A green route line draws on the map from your location to the parking spot
  5. A turn-by-turn directions panel appears showing distance, duration, and step-by-step instructions
  6. To cancel and pick a different spot, tap the X button on the bottom sheet

4.3 Reserving a Spot

  1. Select a parking spot from search or map
  2. Tap "Reserve now"
  3. Choose duration: use the slider (1-8 hours) or quick buttons (1h, 2h, 3h, 4h)
  4. Select payment method:
    • ParkmeWallet — Instant, uses your wallet balance
    • TeleBirr — Mobile money payment
  5. Optional: Enter coupon code PARKME20 for 20 ETB off
  6. Review total cost (price × hours - discount)
  7. Tap "Confirm booking"
  8. Confirmation screen shows:
    • Booking reference (e.g., PR-K3X-729)
    • Space label (e.g., B 27)
    • Arrival time and duration
    • Gate code (tap to reveal, tap to copy)

4.4 During Parking

  • Active timer — The "Parking Active" card shows elapsed time in HH:MM:SS
  • Gate code — Tap to reveal, tap to copy to clipboard
  • Check-in — Mark your arrival to start the active timer

4.5 Managing Your Wallet

  • Open profile drawer → Wallet tab
  • View current balance (top card)
  • Quick top-up: 50, 100, 250, or 500 ETB
  • Select provider: TeleBirr or Commercial Bank of Ethiopia
  • Transaction history shows all deposits (+) and charges (-)

4.6 Viewing History

The History tab shows all past parking sessions with date, location, duration, amount, and status.

5. Host Guide

5.1 Becoming a Host

Hosts are parking space owners who list their spaces on Parkme Ethiopia and earn revenue from driver bookings.

  1. Create a regular driver account (or log in to existing account)
  2. From the sidebar, click "Become a host" or navigate to the Host Dashboard
  3. Your account is immediately upgraded to host status (isHost = true)
  4. Access the Host Dashboard from the profile drawer
Demo host account: Email host@parkme.et / Password demo1234

5.2 Host Dashboard

The dashboard provides a complete overview of your parking business:

Revenue Metrics

MetricDescription
Host PayoutYour 85% share of total earnings
Platform FeeParkme's 15% commission
Total BookingsCount of all bookings across your spaces
Guest RatingAverage rating across all your spaces
Occupancy RatePercentage of spots currently occupied

Weekly Earnings Chart

A bar chart showing earnings for the last 7 days (Mon-Sun), helping you identify peak revenue days.

Space Management

Each of your listed spaces shows:

  • Current price (ETB/hr) with +/- adjustment buttons
  • Available / Total spots
  • Active status toggle (on/off)

Price Adjustment

  • Minimum: 10 ETB/hr
  • Maximum: 500 ETB/hr
  • Changes take effect immediately for new bookings

Availability Management

Block specific dates when your parking space is unavailable:

  • Select a parking space
  • Choose a date
  • Add an optional reason (max 120 characters)
  • Blocked dates prevent new reservations for that day

7.5 Constraints & Assumptions

Constraints

  • GraphHopper free tier: 1,000 directions requests/day
  • Neon PostgreSQL: serverless with connection pooling limits
  • Vercel Hobby plan: 10-second function execution timeout
  • No real payment integration (TeleBirr/CBE are simulated)
  • No real-time WebSocket updates (polled via REST)

Assumptions

  • Users have modern browsers (Chrome 90+, Safari 14+, Firefox 88+)
  • GPS/Geolocation available on user devices
  • Internet connectivity required for map tiles and API calls
  • All parking spaces are pre-seeded; no self-service listing by hosts yet
  • Currency is Ethiopian Birr (ETB); all prices are integers

8.3 API Reference

Authentication

POST /api/auth/signup

Create a new user account

// Request
{ "email": "user@example.com", "password": "secret123", "name": "John Doe", "role": "driver" }

// Response 201
{ "user": { "id": "uuid", "email": "user@example.com", "name": "John Doe", "isHost": false } }
POST /api/auth/login

Authenticate and create session

// Request
{ "email": "user@example.com", "password": "secret123", "role": "driver" }

// Response 200 + Set-Cookie: parkme_session=...
{ "user": { "id": "uuid", "email": "user@example.com", "name": "John Doe", "isHost": false } }
GET /api/auth/me

Get current user from session cookie

// Response 200 (authenticated)
{ "user": { "id": "uuid", "email": "...", "name": "...", "phone": "...", "isHost": true, "createdAt": "..." } }

// Response 401
{ "user": null }
POST /api/auth/logout

8.8 OSM Data Pipeline

Extraction Script

# scripts/extract_addis_osm.py
# Extracts roads and buildings from Geofabrik Ethiopia PBF

# Input:  temp/ethiopia-latest.osm.pbf (132.4 MB)
# Output: public/osm-roads.geojson    (1,294 KB, 5,750 features)
#         public/osm-buildings.geojson (3,179 KB, 16,796 features)

# Parameters:
# Bounding box: lat 8.90-9.05, lon 38.70-38.87 (Addis Ababa)
# Road types: motorway, trunk, primary, secondary, tertiary, residential
# Coordinate precision: 4 decimal places
# Max nodes per way: 40 (simplification)
# Min building area: 250 sq meters

# Run:
pip install osmium
python scripts/extract_addis_osm.py

Optimization Details

  • Compact JSON separators ((",",":")) reduce file size
  • Short property keys ("c" for class, "n" for name)
  • 4 decimal coordinate precision (~11 meter accuracy)
  • Node reduction: max 40 nodes per way
  • Small building filtering: skip <250 sqm
  • Unnamed minor roads skipped
  • Total size: ~4.5 MB (serves well over CDN)

10. FAQ

Q: How do I reset the demo database?

The database auto-seeds on first access when empty. To reset, delete all rows from the users table (cascading to sessions, bookings, wallet_transactions). On next API call, ensureParkmeSeeded() recreates demo data.

Q: How do I add new parking spaces?

Add entries to src/db/seed.ts in the ensureParkmeSeeded() function, then clear the database to trigger re-seeding. In production, spaces are added via direct database access.

Q: Can I run this without GraphHopper?

The map will load without GraphHopper tiles (using fallback), but directions/routing will fail. Set NEXT_PUBLIC_GRAPHHOPPER_API_KEY to enable full functionality.

Q: How do payments work?

Payments are simulated. The wallet system works in-app (balance deduction), but TeleBirr and CBE integrations are placeholder flows. No real money is transferred.

Q: What are the GeoJSON files for?

osm-roads.geojson and osm-buildings.geojson provide detailed OpenStreetMap data for the Addis Ababa area, overlaid on the map for visual context. They are extracted from OSM PBF data using scripts/extract_addis_osm.py.

Q: How do I update the OSM data?

Download the latest ethiopia-latest.osm.pbf from Geofabrik, place in temp/, then run python scripts/extract_addis_osm.py. The updated GeoJSON files will be in public/.
