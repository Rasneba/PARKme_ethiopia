# Parkme Ethiopia

A highly polished, mobile-first, responsive web application and Progressive Web App (PWA) designed to solve the parking challenges in Addis Ababa, Ethiopia.

## Core Features

- **Mobile Mockup & Edge-to-Edge Design**: Implemented with responsive simulated safe area margins (Android 15+ edge-to-edge styling, gesture pill and notch layouts).
- **Interactive Map Engine**: Real-time interactive maps of Addis Ababa landmarks (such as Unity Park, Bole Road, Meskel Square, and National Palace) with active parking pins, you-are-here indicator, and satellite view toggles.
- **Dynamic Reservations**: Book a parking space with customizable hourly durations, promo coupons (e.g. `ARKRIDE20`), and local wallet payments (including telebirr and ParkWallet simulation).
- **Active Passes & Passcodes**: Access digital parking ticket passes with dynamic gate lock/unlock passcodes, ticket notches, and progress-tracking elapsed timers.
- **Unified Landing Page**: High-contrast, beautifully structured homepage showcasing why Parkme exists, how it works, coverage areas across Addis Ababa, and testimonials from local drivers and hosts.
- **Local Persistence & DB Connectivity**: Prepared for Postgres and node-postgres drizzle integration.
